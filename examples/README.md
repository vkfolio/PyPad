This folder contains example Python scripts to test PythonPad features.

## Examples

### 01-hello-world
Simple "Hello World" example to verify PythonPad is working.

### 02-calculator-args
Calculator that takes command-line arguments.
- In the **Args:** field, enter: `10 5`
- Then click **‚ñ∂ Run**

### 03-multi-file
Demonstrates importing local Python modules.
- Opens `main.py`, `utils.py`, and `data_processor.py`
- Shows how files can import from each other

### 04-json-operations
Working with JSON data in Python.

### 05-error-handling
Exception handling and try/except blocks.

### 06-list-comprehensions
List comprehensions, loops, and dictionary comprehensions.

### 07-oop-classes
Object-oriented programming with classes.
- Demonstrates importing custom classes
- Uses `Person` and `Calculator` classes
